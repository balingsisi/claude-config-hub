## 💡 技术决策记录

### 决策 #001: 技术栈选择

**日期**: 2026-03-01
**决策者**: AI Assistant

**决策内容**:
- 框架: Next.js 14 (App Router)
- 语言: TypeScript 5.9+
- 样式: Tailwind CSS 3.4
- 组件: shadcn/ui
- 测试: Vitest + Playwright
- 部署: Vercel

**理由**:
- Next.js 14 App Router 提供最佳性能和开发体验
- TypeScript 保证类型安全
- Tailwind CSS 快速开发，易于定制
- shadcn/ui 提供高质量可定制组件
- Vercel 提供零配置部署

**影响**:
- 开发效率提升
- 学习曲线适中
- 社区支持良好

### 决策 #002: 设计系统

**日期**: 2026-03-01
**决策者**: AI Assistant

**决策内容**:
- 主色调: Blue (221.2 83.2% 53.3%)
- 字体: Inter (Google Fonts)
- 主题: 支持亮色/暗色模式
- 组件风格: 简洁现代

**理由**:
- Blue 是科技感强的颜色
- Inter 是优秀开源字体
- Dark mode 是开发者必需
- 简洁风格更专业
