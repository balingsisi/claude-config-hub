## 📋 使用说明

### 如何使用本清单

```yaml
✅ 已完成: 标记为完成
⏳ 进行中: 标记为进行中
⏸️ 待开始: 等待开始
❌ 已取消: 不再执行
📝 备注: 添加注释
```

### 每周检查流程

```yaml
周一:
  - [ ] 回顾上周任务
  - [ ] 更新任务状态
  - [ ] 规划本周任务
  - [ ] 估算时间

周三:
  - [ ] 检查进度
  - [ ] 调整优先级

周五:
  - [ ] 总结本周完成情况
  - [ ] 规划下周任务
```
